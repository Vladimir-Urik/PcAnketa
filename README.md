# PcAnketa

Simple Minecraft plugin for running player polls with rewards.

Originally developed for the Czech Minecraft server **PvPCraft.cz** as a lightweight system for asking players questions and rewarding them for participation.

⚠️ This project was created as an experimental/internal tool and may require adjustments before production use.

---

## Features

- Create in-game polls for players
- Reward players for completing polls
- Lightweight and simple implementation
- Designed for small to medium Minecraft servers

---

## Use Case

The plugin was designed to allow server administrators to easily gather feedback from players directly in-game.

Example uses:

- voting on server events
- gameplay feedback
- community decisions
- seasonal polls

Players receive a reward after submitting their answer, encouraging participation.

---

## Status

This project was originally written as a custom plugin for **PvPCraft.cz** and has not been extensively tested outside that environment.

Contributions, improvements, and fixes are welcome.

---

## Installation

1. Build the plugin using Gradle:

`./gradlew build`

2. Copy the generated `.jar` file into your Minecraft server `plugins/` folder.

3. Restart the server.

---

## Compatibility

This plugin was developed for a Bukkit/Spigot-based Minecraft server.

Compatibility with newer Minecraft versions is not guaranteed.
